# C# Binary Search Tree (BST) Implementation

A simple **C# console application** demonstrating a **Binary Search Tree (BST)**.  
Educational project for practicing **Algorithms and Data Structures** concepts in C#.

## Features

- `BNode` and `BTree` classes  
- Add nodes while maintaining BST rules  
- Remove nodes with all standard cases handled  
- Print the tree structure  
- Find a node by value  
- Recursive tree traversal (`Walk`)
